# CookDex Brand Color System

## Brand Personality

CookDex should feel:

- warm
- confident
- modern
- tool-like
- practical

Avoid:

- pastel-heavy palettes
- bright neon accents
- cool blue-heavy UI themes

## Core Brand Colors

### Primary Accent

- Name: Flame Orange
- Hex: `#F47A2A`
- Usage: primary actions, active navigation, key highlights

### Pressed / Depth Accent

- Name: Deep Ember
- Hex: `#C44E1F`
- Usage: pressed button state, focused action emphasis

## Surface and Background

### Light Mode

- App background: `#FFF6EC` (Warm Cream)
- Card surface: `#FFFFFF`
- Surface border: `#F1E5DA`

### Dark Mode

- App background: `#1E1A17` (Charcoal Hearth)
- Card surface: `#2A2421`
- Elevated surface: `#342D29`

## Text Colors

### Light Mode

- Primary text: `#2B2B2B`
- Secondary text: `#6F635A`

### Dark Mode

- Primary text: `#F5F2EF`
- Secondary text: `#B8AEA7`

## Functional Colors

- Success: `#6C9A6C`
- Warning: `#E6A23C`
- Error: `#C75B4B`

## UI Rules

- Use orange intentionally, not everywhere.
- Keep interface mostly neutral with warm surfaces.
- Prefer flat surfaces over gradients for core UI panels.
- Keep strong contrast for readability in both themes.
